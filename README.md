# Spatial Clustering of Cultural Heritage Sites in Bosnia & Herzegovina

This project applies the DBSCAN (Density-Based Spatial Clustering) algorithm to geographic data to analyze how changing the search radius (Epsilon) and the minimum points requirement (MinPts) affects the grouping of heritage and tourism sites across Bosnia & Herzegovina.

---

## Project Structure

* **`app.py`:** The main interactive Streamlit application. It contains the code for the web dashboard where users can adjust sliders to change the clustering parameters and view site details using a dropdown menu underneath the map.
* **`notebooks/`:** Contains the Jupyter Notebook (`notebook.ipynb`). This file is used for the step-by-step data analysis, calculating validation scores (Silhouette scores), and running the border proximity checks.
* **`src/`:** Contains the background Python scripts that keep the project organized:
  * `data_processing.py`: Handles loading the data and converting kilometer distances into coordinate math (radians) for the algorithm.
  * `clustering_engine.py`: Contains the core function that runs the DBSCAN model.
* **`data/`:** Holds the primary dataset file (`heritage_sites_bih.csv`).
* **`bih_integrated_density_dashboard.html`:** A standalone map webpage generated by the notebook. It saves a pre-rendered view of the clusters that can be opened in any browser without running Python.

---

## Dataset Source

The dataset tracking 272 unique historical landmarks across the country was specifically compiled, structured, and aggregated from primary regional records. The finalized dataset was subsequently published and hosted on Kaggle:

* **Dataset Link:** [Bosnia and Herzegovina Heritage Landmarks Dataset](https://www.kaggle.com/datasets/danakopi/bosnia-and-herzegovina-heritage-landmarks-dataset)
* **Data Fields Used:** Site Name, Municipality Location, Category, Latitude, Longitude, Description, and Historical Facts.---

## How to Run the Project Locally

To run the interactive dashboard on a local machine, follow these steps:

1. Clone the repository and navigate into the project folder:

```bash
   git clone https://github.com/dzankk/geo-tourism-clustering-dbscan.git
   cd geo-tourism-clustering-dbscan
```

2. Install the required data science and mapping libraries:

```bash
   pip install pandas numpy scikit-learn folium streamlit streamlit-folium
```

3. Launch the local Streamlit application server:

```bash
   streamlit run app.py
```
